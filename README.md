## ----------------Project_online----------------

### Check Version Python
```
python --version
```

### Create Environment
```
python -m venv env
```

### Activate Environment
```
env\scripts\activate
```
### Install Lib
```
pip install -r requirements.txt
```

### Django migrate Model
```
python manage.py makemigrations
python manage.py migrate
```

### Start Server Django
```
python manage.py runserver
```
